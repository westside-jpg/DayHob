# DayHob 📸

> Хватит думскроллить и смотреть на чужую жизнь. Пора жить свою!

DayHob помогает раскрыть себя и найти дело по душе. Новый день - новое задание для всех пользователей. Пробуй, делись, находи своё хобби.

## Стек
**Backend:** `FastAPI` `PostgreSQL` `SQLAlchemy` `bcrypt` `JWT`<br>
**Frontend:** `Jinja2` `HTML` `CSS` `JavaScript`
